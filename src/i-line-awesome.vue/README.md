## 使用方法

### 在 CSS 中使用（推荐）

只引用具体的图标，可以减少最后的打包大小。

``` css
.bar::before {
    icon-font: url('i-line-awesome.vue/assets/500px.svg');
}
```

### 在 Vue 组件中使用

需要先引入包

``` js
import ILineAwesome from 'i-line-awesome.vue';

Vue.component(ILineAwesome);
```

然后按照如下方式使用

``` html
<i-line-awesome name="500px"></i-line-awesome>
```
